# Replicating Progress Measures for Grokking via Mechanistic Interpretability
Recreation and further experimentation of Nanda et al.'s Progress Measures For Grokking Via Mechanistic Interpretability. This uses their later provided [Grokking_Demo Colab notebook](https://colab.research.google.com/github/neelnanda-io/TransformerLens/blob/main/demos/Grokking_Demo.ipynb) rather than their [original notebooks on GitHub](https://github.com/mechanistic-interpretability-grokking/progress-measures-paper).

Paper which this repo is based on: https://doi.org/10.48550/arXiv.2301.05217
